--------
{: data-content=" 难点 " }

- 拍扫弦的音量控制
- 节奏的稳定
- 延音的处理

--------
{: data-content=" 需要加强的 " }

- 三连音节奏不是很准
- 最后几个音没有弹奏响(太遗憾了。。。)
- 一些根音不是很连续，一些使用闷音，一些使用的延音

--------
{: data-content=" 做的不错的 " }

- 整体比较连贯, 比较干净
- 弹奏的时候相对放松（跟之前比起来) 尽量地多看看镜头
- 反复听自己的录音, 并找到自己不足的点(比如在某一些点的音量过高或者过低)
- 在练习的时候开着节拍器, 在录音的时候开着耳返
